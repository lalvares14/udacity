Basics of the web and HTML

The web

The web is basically a network of different computers interacting with each other. To comunicate they stabilished a common language, to facilitate the interaction. The protocol chosen for this communication was HTTP, which is used to all types of communication on the web, specially requests made by web browsers.

The video helps a lot explaining these concepts.

Html

Hipertext Markup Language is the language of the web, all website have some html. It is the language used to write content for the web, and also managing the impression of this content by the final user.

elements and tags

HTML is a mix between elements for content and tags for design, it is also very strict comparing to day-a-day language, not admitting typos or other mistakes without risking misunderstandings with the computer.

inline and blocks

There are also inline and block elements in html, which help to structure any webpage and disign. Inline is more related to text styling, while block is more related to structuring elements in a page.
